# HwGDReqs integration

This mod is not standalone, requires [HwGDReqs](https://hwgdreqs.github.io) to work (it is still in pre-release for now)

### So what is even HwGDReqs?
a GD level requests app for streamers, queue based, with both Youtube (Username alone is needed) and Twitch support (Device flow login), with filters (difficulty, lengh, disabling dislikes), ability to blacklist a level / creator / requester, twitch commands (!del levelid) (!replace levelid newlevelid

Twitch-only features:
- !queue chat command to reply as you with queue contents (optional)
- !whereami chat command to reply as you with the requester's level position (optional)
- Ban Requester button (Both in desktop app and this mod) to ban the requester from your chat (optional)

### As if v0.2.0:
- Now you can customize API port in both the desktop app and this mod (in case you're hosting smth else in port :6767)
- Also you can run the desktop app on your desktop, and the mod on your phone to receive levels on ur phone! (Local network)
